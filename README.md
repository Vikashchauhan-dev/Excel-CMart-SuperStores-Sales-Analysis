# 🛒 CMart Stores Sales Analysis Dashboard

## Project Overview

The CMart Stores Sales Analysis Dashboard is an interactive Excel-based business intelligence project designed to monitor sales performance, profitability, customer behavior, product trends, and regional performance.

The dashboard provides meaningful insights into sales data from **2021 to 2026**, helping stakeholders make data-driven business decisions.

---

# Problem Statement

Retail businesses generate large volumes of transactional data, making it difficult to identify sales trends, profitable products, customer preferences, and regional performance.

The objective of this project is to transform raw sales data into an easy-to-understand dashboard that enables users to:

- Track overall business performance
- Monitor monthly sales trends
- Identify top-selling products
- Analyze customer ratings
- Compare regional sales performance
- Evaluate discounts and profitability
